# 🛒 E-Commerce Data Warehouse — SQL Project

This is my SQL project where I built a small data warehouse for an e-commerce system using MySQL.

I created tables like customers, products, orders, etc., and then wrote queries to analyze data like revenue, customer behavior, and product performance.

---

## 🚀 How to Run

1. Open MySQL Workbench and connect to your server.

2. Open `01_schema.sql` and run it.
   → This creates database and tables.

3. Open `02_seed_data.sql` and run it.
   → This inserts sample data.

4. Open `03_analytics.sql`
   → Run queries one by one to see results.
